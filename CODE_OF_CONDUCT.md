# Code of Conduct

Dự án yêu cầu trao đổi chuyên nghiệp, minh bạch và tập trung vào bằng chứng.

## Expected behavior

Tôn trọng, phản biện vào thiết kế hoặc bằng chứng, không công kích cá nhân, không che giấu rủi ro hoặc trạng thái chưa hoàn thiện.

## Enforcement

Owner có quyền từ chối thay đổi vi phạm governance, security hoặc scope freeze.
