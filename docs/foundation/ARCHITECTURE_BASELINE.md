# Architecture Baseline

Web/PWA là giao diện chính; Operations Ledger là source of truth; CVF là control layer; Refinery làm sạch; providers là replaceable capabilities.

## Frozen principles

Provider-neutral, rules-first, LLM-assisted, human-approved, evidence-traceable, degraded-mode capable.

## Change

Mọi thay đổi module boundary hoặc trust boundary cần Architecture Amendment.
